# memo
本地-文字记录存档系统；暂不支持图片。可导出/导入txt文件。

#### 介绍
本地：文字记录存档
可导出/导入txt文件

#  注意：清除浏览器数据前先导出记录到本地

1.  记录的数据默认存储于浏览器内部数据
2.  清除浏览器缓存与"Cookie和其他网站数据"会丢失数据
3.  手机端导出功能可能受限，取决于浏览器功能是否限制。
